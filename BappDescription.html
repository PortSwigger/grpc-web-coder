gRPC-Web Coder enhances your ability to analyze and manipulate gRPC-Web traffic by automatically decoding and encoding payloads. It supports both application/grpc-web-text and application/grpc-web+proto content types, streamlining the process of inspecting and modifying gRPC-Web messages during security assessments.

Features

  • Automatic detection and decoding of gRPC-Web payloads based on Content-Type or x-grpc-content-type headers.
  • Support for both application/grpc-web-text and application/grpc-web+proto formats.
  • Dedicated tab for viewing and editing decoded protobuf messages directly within the message editor.
  • Seamless re-encoding of modified messages back into the appropriate gRPC-Web format.

Usage

Once the extension is active, it automatically processes requests with the appropriate Content-Type or x-grpc-content-type headers. The decoded protobuf message is displayed in a dedicated tab within the message editor, allowing for easy inspection and modification. Any changes made are automatically re-encoded when the message is sent.

Configure

For manual control, you can enable or disable the decoding features using the provided checkboxes in the extension's settings tab. This flexibility allows you to tailor the extension's behavior to your specific testing needs.
